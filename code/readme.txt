Le fichier a besoin de "pse.h" pour fonctionner. La libraire stdbool a été ajoutée à ce dernier.

Le code est composé de 2 fichiers :
	- client.c s'agissant du programme pour le client
	- serveur.c s'agissant du programme pour le serveur

Pour compiler le programme, il suffit de faire un "make" dans le dossier code de ce projet.


But du projet : Ce projet est une version très simplifiée du site "Omegle" permettant de se discuter aléatoirement avec des inconnus. 
		Une fois connecté à un autre client, deux commandes sont possibles : 
			- NEXT! pour passer à la personne suivante 
			- FIN! pour terminer la session
